# termux-proot

https://github.com/Yonle/termux-proot

Note that 2.0.0 may not work
Install:
```
curl -sLo ~/termux-proot.sh https://github.com/Yonle/termux-proot/raw/v1.0.0/termux
chmod +x ~/termux-proot.sh
~/termux-proot.sh
```
Run:
`~/termux-proot.sh`
Remove:
```
rm termux-proot.sh
proot -0 rm -rf ~/.termux-fs
```
